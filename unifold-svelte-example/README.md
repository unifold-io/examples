# Unifold Svelte Example

A SvelteKit app demonstrating the [`@unifold/connect-svelte`](https://www.npmjs.com/package/@unifold/connect-svelte) SDK.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file with your Unifold publishable key:

```
PUBLIC_UNIFOLD_KEY=pk_live_YOUR_KEY_HERE
```

3. Start the dev server:

```bash
npm run dev
```

The app will be available at [http://localhost:5173](http://localhost:5173).
